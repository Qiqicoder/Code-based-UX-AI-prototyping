# Selected work

A single static page. No build step. Factual listing of four projects for a hiring manager.

## Structure
- `index.html` — the page (open directly to preview)
- `assets/` — Reply hero image, Talk to Grow video + poster, Reply spec (.md)
- `demos/` — two Reply interactive prototypes (open standalone)

## Deploy to Vercel (zero config)
Option A — dashboard: drag this whole folder onto vercel.com/new.
Option B — CLI: `npm i -g vercel` then run `vercel` in this folder, accept defaults.
Option C — GitHub: push this folder to a repo, then "Import Project" on Vercel.

Static HTML at the root deploys as-is; no framework or settings needed.
